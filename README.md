# 1 - (Sugerido) Crear entorno virtual

Crear venv:

```
python -m venv nombre_entorno
```

Activar entorno (Windows):

```
carpeta_venv/Scripts/./Activate.ps1
```

Desactivar entorno virtual:

```
deactivate
```
